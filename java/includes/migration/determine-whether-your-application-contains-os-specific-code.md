### <a name="determine-whether-your-application-contains-os-specific-code"></a>判斷您的應用程式是否包含作業系統特定的程式碼

如果您的應用程式包含任何在主機作業系統上具有相依性的程式碼，則您必須重構它以移除這些相依性。
